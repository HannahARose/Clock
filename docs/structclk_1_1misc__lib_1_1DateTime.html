Detailed Description

Class representing a point in time.

This class encapsulates a time point using boost's posix_time library and provides methods to manipulate and retrieve time information.

DateTime clk::misc_lib::DateTime::fromISO ( const std::string &  iso_string)
static

Factory method to create a DateTime object from an ISO 8601 string.

Parameters
iso_stringThe iso formatted string to parse.
Returns
A DateTime object representing the parsed time.
Exceptions
std::invalid_argumentif the iso_string is empty or invalid.

This method uses boost's posix_time to parse the ISO string. It assumes the string is in the format "YYYY-MM-DDTHH:MM:SS" Followed by a time zone suffix:

  • "" for Local time
  • "Z" for UTC
  • "+HH:MM" or "-HH:MM" for custom offsets.
