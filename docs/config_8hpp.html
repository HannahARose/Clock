enum class clk::si3_sim::RunSchedule : std::uint8_t
strong

Enum for the run schedule types.

This enum defines the different types of run schedules that can be used in the simulation. Primary use is to determine the periodicity of the simulation run schedule.

Enumerator
DAILY 

Same schedule every day.

WEEKLY 

Same schedule every week.

MONTHLY 

Same schedule every month.

MJD 

Same schedule for every Modified Julian Date modulo a given value.

YEARLY 

Same schedule every year.
